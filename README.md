Este projeto foi construído com foco em performance e design moderno utilizando:

- **[TypeScript](https://www.typescriptlang.org/)**

- **[Radix UI](https://www.radix-ui.com/)** para componentes acessíveis

## 🚀 Como rodar localmente

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado na sua máquina.

1. Clone o repositório (após subir para o GitHub):
   ```bash
   git clone https://github.com/SEU-USUARIO/seu-repositorio.git
   cd seu-repositorio
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Abra o seu navegador em `http://localhost:5173`.

## 📦 Como fazer o Build

Para gerar os arquivos otimizados para produção:

```bash
npm run build
```

## 🌐 Deploy

Este projeto pode ser hospedado em plataformas como **Vercel** ou **Cloudflare Pages**.

- **Na Vercel:** Importe o repositório, coloque o *Build command* como `npm run build` e o *Output directory* como `dist/client`.
- **No Cloudflare Pages:** Importe o repositório e deixe o comando de build como `npm run build` (ele detectará automaticamente o resto).
